jalankan stack

download zip enterprise
upload ke volume mnt/extra/addons
extract

edit /etc/odoo/odoo.conf
addons_path = /mnt/extra-addons/enterprise-16.0
; jika mau pakai ssl
web_base_url_freeze = True
proxy_mode = True

pakai traefik ya untuk ssl-nya
atau pakai nginx proxy manager, tambahkan advanced config di host > proxy hosts > edit > advanced
proxy_redirect http://yourdomain/ https://yourdomain/; 
agar sewaktu konfigure website bisa menggunakan nama domain,
karena kalau gak pakai ini, terjadi mixed content http tanpa s yang mengakibatkan diblokir sama browser

restart container
